# StoryWeaver Backend

Backend API sviluppato in Laravel per StoryWeaver, un progetto dedicato alla creazione e gestione di storie interattive a nodi.

## Tecnologie usate

- PHP
- Laravel
- MySQL
- Laravel Sanctum
- API REST
- Storage per immagini e audio

## Funzionalità principali

- Autenticazione utente
- CRUD delle storie
- CRUD dei nodi narrativi
- CRUD delle scelte
- CRUD dei token
- Upload immagini per i nodi
- Upload audio ambientale per le storie
- Gestione del nodo iniziale
- Creazione multipla di nodi, scelte e token
- API pensate per essere consumate da un frontend React
